# Portfólio - Daniel

Site de portfólio pessoal desenvolvido para a faculdade, usando HTML, CSS e
JavaScript puros (sem frameworks). Apresenta quem eu sou, meus projetos e uma
página de contato.

**Site no ar:** https://danielbezerracnc.github.io/DanielPortifolio/
(disponível após ativar o GitHub Pages em Settings → Pages — confira o link depois)

## Páginas

O site tem três páginas, com navegação entre elas pelo menu:

- `index.html` — página inicial, com foto e apresentação
- `projeto.html` — lista dos meus projetos
- `contato.html` — formulário de contato

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Critérios atendidos

- **Layout** bem estruturado com **Flexbox** (menu) e **CSS Grid** (cards de projetos)
- **Tags semânticas**: `header`, `nav`, `main`, `section`, `article`, `footer`
- **Navegação simples** entre 3 páginas (index + duas outras)
- **Recursos em JavaScript**:
  - botão que troca entre tema claro e escuro
  - validação do formulário de contato
- **Design responsivo** (se adapta à tela do celular)
- **Marcação válida** segundo o [W3C Markup Validation Service](https://validator.w3.org/)

## Arquivos do projeto

- `index.html`, `projeto.html`, `contato.html` — as páginas do site
- `styles.css` — todo o estilo (cores, layout, responsividade)
- `script.js` — o JavaScript (tema e validação)
- `foto.jpeg` — minha foto

## Como ver o site

Abra o arquivo `index.html` no navegador, ou use a extensão **Live Server** no
VS Code.
