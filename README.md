# Portfólio - Daniel

Meu site de portfólio, feito com HTML, CSS e JavaScript para a faculdade.

## Arquivos
- `index.html` - página inicial
- `projeto.html` - meus projetos
- `contato.html` - formulário de contato
- `styles.css` - todo o estilo do site
- `script.js` - o JavaScript (tema e validação)

## O que o site usa
- Layout com **Flexbox** (menu) e **CSS Grid** (projetos)
- Tags semânticas: header, nav, main, section, article, footer
- **JavaScript**: botão que troca o tema (claro/escuro) e validação do formulário
- Responsivo (funciona no celular)

## Como ver
Abra `index.html` no navegador, ou use a extensão Live Server no VS Code.
